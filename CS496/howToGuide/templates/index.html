Google Charts for Android Applications

What This Guide Covers

This guide will lead you through the basics of generating data visualizations for your Android applications using the Google Charts API. We will begin by illustrating how to create and customize a simple pie chart. Next, we will explore feeding data received from user input and HTTP requests to your charts. Finally, we will look at some of the more advanced features the Google Charts API offers, like chart animation and controls.

Prior Knowledge

A basic understanding of the following will be crucial to getting the most out of this guide:

  • Java
  • JavaScript
  • HTML
  • Android Studio
  • Asynchronous HTTP Requests
  • JSON
