Preparing Assets

Now that we've prepared our WebView for handling JavaScript code, we have to load the Google Charts API. To do this, we are going to create an HTML file that will be responsible for displaying the chart by calling two scripts - one to load the API and another to actually draw the chart. Let's begin by creating an assets folder to our project structure as doing so will allow us to programtically access the resources we place within this directory. Right click the app directory in your project view, select New, then Folder and finally click Asset Folder. Keep the default settings presented by the resulting dialog and click the Finish button. The assets folder should now be present in your project structure. Below are a few screen shots illustrating the process:

Creating the Folder


The New Android Component Dialog


The Project View


Note that when the asset folder is created it will not have the html and js subfolders shown in the last of the previous images. I created these to facilitate better project organization. To mimic this, all you need to do is right click the assets folder now present in your project view, select New and then click the Directory option.
