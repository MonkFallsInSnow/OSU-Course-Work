Smallsh
-------------------------

Program Files:
smallsh.c
smallsh.h
main.c
makefile

To Compile:
1) Make sure all the program files are in the same folder
2) Type "make" (no quotes) into the command prompt. Alternatively, you could type gcc -o smallsh main.c smallsh.c

To Run:
1) Type "smallsh" (no quotes) into the command prompt
