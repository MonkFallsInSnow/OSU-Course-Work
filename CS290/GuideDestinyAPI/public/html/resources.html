The Black Garden

I hope what you have learned by reading this guide will prove useful as you endeavor to create your own Destiny apps. Below are several links to various wikis, forums and the official documentation. Working with this API can be a bit of a struggle, but fortunately there is large community of helpful coders centered on the Bungie forums willing to help, and I encourage you to explore that resource, post your questions and contribute when you can. Good luck and happy coding!


Resources

Node.js

  • Express.js - documentation for the express.js module
  • Request.js - documentation for the request.js module
  • Body-Parser.js - documentation for the body-parser.js module
  • Handlebars.js - documentation for the handlebars.js module

Destiny API

  • Bungie Forum - your main resource for getting help with API
  • Official Introduction of the API - an overview of the API's basic requirements
  • General Endpoints - an official list of available URIs for general and character related queries
  • Forum Endpoints - an official list of available URIs for forum related queries
  • User Account Endpoints - an official list of available URIs for user account related queries
  • Unofficial Wiki - a community supported wiki on the API(the search bar is your friend here)
  • More Endpoints - an unofficial list of endpoints
  • Manifest Definition Types - an unofficial list of type codes
  • Query String Parameters - an unofficial list of query string key value pairs
  • Authentication - a basic overview of the user authentication process and some related scripts
