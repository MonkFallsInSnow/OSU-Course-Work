Your Key to The City

Most of the requests you can make with the Destiny API require three important components: an API key, a membership ID and a character ID. We will cover how to get both your membership and character IDâ€™s on the next page, but before either of those crucial pieces of information can be accessed you need to get an API key.



Getting an API key

You can get an API key by visiting https://www.bungie.net/en/User/API and signing in to your Bungie account. All you need to do is follow the instructions on each page, but if you are more of a visual learner, here are a few pictures describing the process:


1) Click the sign in or join up buttons (both lead to the same page).




2) Choose your account platform and click the appropriate link. A login page will pop up, so just enter your credentials.




3) Enter your username, email address and agree to the terms of service, then click on the "GIVE ME THE KEY" button.




4) There you go! You should now have a valid API key. It's very important that you don't share this key with anyone (notice I blocked mine out). Anyone with this key can make requests of the API in your name and there is no way of knowing what kind of activity that might entail. Its better to play it safe.
